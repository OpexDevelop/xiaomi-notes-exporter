Xiaomi Notes Exporter

Где взять файл бэкапа?
1. Настройки ➔ О телефоне ➔ Резервирование и восстановление ➔ Мобильное устройство.
2. Выберите только Заметки в системных приложениях.
3. Файл появится по пути: MIUI/backup/AllBackup/date_time/Notes(com.miui.notes).bak

Перетащите файл .bak сюда или нажмите для выбора
